Privacy Policy for Calc3 Assistant

Effective Date: August 5, 2025

Calc3 Assistant is a browser extension that allows users to perform symbolic multivariable calculus computations. We respect your privacy and are committed to protecting it.

What data do we collect?

We do not collect or store any personal information.

When you use the extension, your mathematical input may be sent temporarily to a secure backend server to compute symbolic results. This input is not stored, logged, or shared with third parties.

Permissions

  • Storage: Used to save your input locally in your browser so it can be restored next time you open the extension.
  • Host permissions: Used to send your math expressions to our backend server for symbolic processing.
  • Active Tab Used to potentially read the current tab URL if future functionality is added to insert math fields into certain web pages.

Data sharing

We do not share your data with any third parties.

Security

All communications between the extension and the backend are encrypted via HTTPS.
